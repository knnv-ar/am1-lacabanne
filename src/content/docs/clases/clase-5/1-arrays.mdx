Un array es una lista de datos. Es posible almacenar muchos valores en una sola variable utilizando este tipo de estructura. Los arrays son muy útiles para manejar grandes cantidades de información de forma organizada.

Cada valor dentro de un array tiene un índice que indica su posición. Es importante recordar que en programación, el primer elemento de un array siempre tiene el índice 0.

### 1.1. Propiedades de los arrays

Cada array puede darnos información sobre su composición. De esta forma si accedemos a la propiedad `length`, el array nos devolverá el número de elementos que posee.

### 1.2. Declaración y asignación

Usar arrays es similar a trabajar con variables únicas, sigue los mismos patrones. Como ya sabés, con este código puedes crear una sola variable llamada `x`:

```javascript
let x;
```

Para crear un array, basta con definir que el valor de la variable sea un par de corchetes vacíos:

```javascript
let x = [];
```

Observá que no es necesario declarar por adelantado la longitud del array, la longitud es determinada por el número de elementos que vos pones en él.

Para asignar valores a un array, podemos hacerlo de forma individual utilizando el índice:

```javascript
x[0] = 10;
x[1] = 20;
x[2] = 30;
```

O podemos asignar todos los valores al momento de la declaración:

```javascript
let x = [10, 20, 30];
```

### 1.3. Acceder a los valores

Para utilizar un valor de un array, escribimos el nombre de la variable seguido del índice entre corchetes:

```javascript
line(x[0], 0, x[0], height);
```

### 1.4. Arrays bidimensionales

Hasta ahora hemos trabajado con arrays de una sola dimensión. Sin embargo, a veces necesitamos estructuras más complejas para organizar nuestros datos. Imaginemos que queremos guardar coordenadas X e Y de varios puntos. Podríamos tener dos arrays diferentes:

```javascript
let x = [10, 40, 37];
let y = [30, 20, 82];
```

Esto está bien y funciona, pero existe una mejor forma de solucionarlo que nos permite tener toda la información en un solo lugar. Los arrays bidimensionales son arrays que contienen otros arrays.

De esta forma podremos agregar valores relacionados dentro del array. Básicamente lo que estamos haciendo es agregar un array dentro de otro. La sintaxis sería la siguiente

```javascript
let xy = [[10, 30],[40, 20],[37,82]];
```

Si quisiéramos agregar un valor nuevo en vez de pasar un número como valor pasariamos el array de la siguiente forma:

```javascript
xy.push([50, 60]);
```

### 1.5. Objetos

Otra forma de organizar información relacionada es a través de objetos. Los objetos nos permiten guardar datos asociados a "llaves" o nombres, en vez de índices numéricos.

Dentro de un objeto se pueden guardar múltiples valores como en un array, pero en vez de acceder a ellos por su posición, lo hacemos por su nombre.

La sintaxis para generar objetos es:

```javascript
let objs = {llave1: 100, llave2: 460, llave3: 17};
```

Acá vemos como creamos el objeto objs e dentro le agremos tres llaves diferentes con diferentes valores. El nombre de las llaves puede ser cualquiera como si fuese una variable más.

Para acceder a los valores de un objeto, usamos el punto seguido del nombre de la llave:

```javascript
ellipse(objs.llave1, objs.llave2, 50, 50);
```

También podemos modificar los valores de la misma forma:

```javascript
objs.llave1 = 20;
```

Es muy común que estos objetos estén dentro de un array. La sintaxis correcta para agregar objetos a un array es la siguiente:

```javascript
let objs = [
  {llave1: 14, llave2: 12.2, llave3: 17},
  {llave1: 3, llave2: 8.9, llave3: 12},
  {llave1: 8, llave2: 11.0, llave3: 99}
];
```
