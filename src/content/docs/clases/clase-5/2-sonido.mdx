p5.js no solo nos permite trabajar con imágenes, sino también con sonido. Para esto, necesitamos incluir la librería `p5.sound.js` en nuestro proyecto (que ya viene incluida en el editor online).

### 2.1. Cargar y reproducir sonidos

Para trabajar con archivos de audio, primero debemos cargarlos en la memoria. Es recomendable hacerlo dentro de la función `preload()` para asegurar que el archivo esté listo antes de que el programa comience.

```javascript
let sonido;

function preload() {
  sonido = loadSound('audio.mp3');
}

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
}

function mousePressed() {
  sonido.play();
}
```

### 2.2. Control de volumen y paneo

Podemos controlar diferentes propiedades del sonido mientras se reproduce:

- **Volumen:** `sonido.setVolume(valor)` (donde valor es entre 0.0 y 1.0).
- **Paneo:** `sonido.pan(valor)` (donde valor es entre -1.0 izquierda y 1.0 derecha).
