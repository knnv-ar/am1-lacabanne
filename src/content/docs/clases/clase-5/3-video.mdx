De la misma forma que con el sonido, p5.js nos permite trabajar con video, ya sea cargando un archivo o capturando la señal de la cámara web.

### 3.1. Captura de cámara

Para usar la cámara web, utilizamos la función `createCapture(VIDEO)`.

```javascript
let captura;

function setup() {
  createCanvas(400, 400);
  captura = createCapture(VIDEO);
  captura.size(400, 400);
  captura.hide(); // Ocultamos el elemento DOM original
}

function draw() {
  background(220);
  image(captura, 0, 0, width, height);
}
```

### 3.2. Cargar un archivo de video

Para cargar un video desde un archivo, usamos `createVideo()`.

```javascript
let video;

function setup() {
  createCanvas(400, 400);
  video = createVideo(['video.mp4']);
  video.hide();
  video.loop(); // Reproducir en bucle
}

function draw() {
  background(220);
  image(video, 0, 0);
}
```
