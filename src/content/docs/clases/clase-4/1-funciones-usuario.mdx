Como ya vimos, las funciones son las unidades básicas de un programa en p5.js. 

Hasta ahora hemos usado dos tipos de funciones:

- Funciones del sistema: propias de p5.js como `createCanvas()`, `line()`, `fill()`, etc.
- Estructuras del sistema: bloques de código como `setup()` y `draw()`.

Sin embargo, también podemos escribir nuestras propias funciones. Escribir funciones propias nos ayuda a organizar el código y a reutilizarlo. 

### 1.1. Ventajas de usar funciones de usuario

- **Modularidad:** Las funciones permiten dividir un programa complejo en partes más pequeñas y manejables.
- **Reutilización:** Una vez que escribimos una función, podemos usarla tantas veces como queramos en nuestro programa.
- **Claridad:** Al poner un nombre descriptivo a una función, el código se vuelve más fácil de leer y entender.

### 1.2. Declaración de una función

Para declarar una función de usuario, se utiliza la palabra reservada `function`, seguida del nombre de la función, paréntesis y llaves. El código que queremos que se ejecute cuando llamamos a la función se pone dentro de las llaves.

```javascript
function nombreDeLaFuncion() {
  // Código a ejecutar
}
```

### 1.3. Llamada a una función

一度 hemos declarado una función, podemos llamarla en cualquier parte de nuestro programa (usualmente dentro de `draw()`) escribiendo su nombre seguido de paréntesis.

```javascript
nombreDeLaFuncion();
```

### 1.4. Funciones con parámetros

Las funciones pueden recibir información externa para realizar su tarea. A esta información se la denomina parámetros. Los parámetros se declaran dentro de los paréntesis de la función.

```javascript
function dibujarCirculo(x, y, diametro) {
  ellipse(x, y, diametro, diametro);
}
```

Al llamar a la función, debemos pasarle los valores para esos parámetros:

```javascript
dibujarCirculo(100, 100, 50);
```

Lo importante a la hora de escribir una función, es tener la idea clara de lo que queremos que haga. Una vez que tenemos esa idea, podemos traducirla a código.

En los ejemplos anteriores, la salida producida por una función dibuja una figura o una forma en la pantalla.

No obstante, hay veces que preferimos que la propia salida que se produce sea un tipo de dato. A esto se lo llama valor de retorno.

Las funciones pueden hacer un cálculo y luego retornar un valor al programa principal. 

Ya hemos usado funciones de este tipo, incluyendo `random()` y `sin()`. Cuando esta función aparece, el valor de retorno es generalmente asignado a una variable:

```javascript
let r = random(1, 10);
```

En este caso, la función `random()` retorna un valor entre 1 y 10, el que luego es asignado a la variable r.

Para hacer que una función retorne un valor, se utiliza la palabra reservada `return`.

```javascript
function sumar(a, b) {
  let resultado = a + b;
  return resultado;
}
```
